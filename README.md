# Homezy

Homezy is a native application for the Apple ecosystem developed in **Swift** and **SwiftUI**. The project aims to simplify home management and improve the quality of domestic life through a modern, intuitive, and accessible interface.

* **Environment management:** Organization and monitoring of rooms in the house.
* **Intuitive interface:** Developed entirely with SwiftUI to ensure fluidity and visual coherence with Apple's guidelines.
* **Language:** Swift
* **UI framework:** SwiftUI
* **Tools:** Xcode
* **Architecture:** MVVM (Model-View-ViewModel)
